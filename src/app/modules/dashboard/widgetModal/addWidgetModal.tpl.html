Date selection

The Date Selection widget is a helper which allows you to specify a date range for a dashboard. The range can be a Day, Week, Month or a specific period.
The following value names can be inserted into dashboard queries (eg. "created = {start}"):

  • {start} - The start of the date range
  • {end} - The end of the date range
